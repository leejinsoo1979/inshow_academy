# 09. 관리자 운영 설계

## 1. 관리자 권한

| Role | 권한 |
|---|---|
| super_admin | 모든 기능, 관리자 계정 관리 |
| admin | 강의/신청/결제/환불/공지 관리 |
| staff | 신청자 조회, 출석/메모, 자료 업로드 일부 |

## 2. 대시보드 지표

- 오늘 신청 수
- 오늘 결제 금액
- 결제 성공률
- 환불 요청 수
- 마감임박 강의
- 정원 대비 신청률

## 3. 강의 관리

- 강의명, slug, 카테고리, 레벨, 가격, 공개 상태
- 대표 이미지 또는 흑백 그래픽 이미지
- 상세 설명, 수강 대상, 커리큘럼
- 일정/회차, 장소, 정원
- 제공 자료

## 4. 신청자 관리

- 이름, 연락처, 이메일
- 신청 강의/회차
- 결제 상태
- 신청 상태
- 관리자 메모
- CSV 내보내기

## 5. 운영 로그

관리자 변경 작업은 audit_logs에 남긴다. 예: 강의 가격 변경, 정원 변경, 환불 승인, 신청 상태 변경.
