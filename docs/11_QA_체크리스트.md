# 11. QA 체크리스트

## 1. 공통 UI

- 모바일 360px, 태블릿, 데스크톱에서 레이아웃 깨짐이 없는가?
- 블랙앤화이트 톤이 유지되는가?
- 이모지형 아이콘이 사용되지 않았는가?
- 버튼/링크 hover/focus/disabled 상태가 있는가?
- 폰트 크기와 줄간격이 가독성을 충족하는가?

## 2. 강의 탐색

- 필터/정렬이 정상 동작하는가?
- 마감/마감임박 상태가 정확히 표시되는가?
- 강의 상세 URL 직접 접근이 가능한가?
- 없는 slug 접근 시 404 또는 안내 페이지가 표시되는가?

## 3. 신청/결제

- 필수 입력값 검증이 되는가?
- 약관 동의 없이는 결제 진행이 막히는가?
- 주문 생성 후 결제창이 정상 호출되는가?
- successUrl에서 paymentKey/orderId/amount가 처리되는가?
- 금액 변조 시 서버가 결제를 거부하는가?
- 중복 클릭/새로고침에도 중복 결제가 방지되는가?
- 결제 실패 시 재시도 경로가 명확한가?

## 4. 마이페이지

- 로그인하지 않은 사용자는 접근이 제한되는가?
- 내 주문/신청만 조회되는가?
- 자료 다운로드 권한이 수강생에게만 허용되는가?

## 5. 관리자

- admin이 아닌 사용자는 접근할 수 없는가?
- 강의 등록/수정/비공개가 동작하는가?
- 신청자 목록 필터/검색이 동작하는가?
- 환불 처리 후 주문/결제/수강 상태가 일관되게 변경되는가?

## 6. 배포 전

- 환경변수 누락 여부 확인
- 결제 테스트 키/실결제 키 분리
- robots.txt, sitemap.xml, OG 태그 확인
- 로그/에러 모니터링 연결
- DB 백업 정책 확인
