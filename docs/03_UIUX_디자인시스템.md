# 03. UI/UX 디자인 시스템

## 1. 디자인 콘셉트

INSHOW ACADEMY의 UI는 '프리미엄 인테리어 스튜디오'와 '실무 교육 플랫폼'의 중간 지점을 목표로 한다. 교육 플랫폼이지만 가볍거나 캐주얼해 보이지 않게 하고, 블랙앤화이트 기반의 절제된 정보 구조를 사용한다.

## 2. 톤앤매너

- 모던함
- 고급스러움
- 실무적 신뢰감
- 과장 없는 전문성
- 넓은 여백과 선명한 대비

## 3. 금지 요소

- 이모지 아이콘
- 과한 컬러 그라데이션
- 둥근 캐릭터형 일러스트
- 불필요한 배너/팝업
- 저가 강의몰 느낌의 강한 할인 UI

## 4. 컬러 토큰

| Token | Value | 사용처 |
|---|---|---|
| `--color-black` | `#050505` | 메인 텍스트, CTA 배경 |
| `--color-white` | `#FFFFFF` | 페이지 배경, 카드 배경 |
| `--color-off-white` | `#F7F7F4` | 섹션 배경 |
| `--color-gray-900` | `#1A1A1A` | 제목/강조 |
| `--color-gray-600` | `#666666` | 보조 텍스트 |
| `--color-gray-300` | `#D9D9D9` | 라인/보더 |
| `--color-gray-100` | `#F1F1F1` | 비활성/배경 |

## 5. 타이포그래피

- 한글: Pretendard 또는 Noto Sans KR 계열
- 영문/숫자: Inter 또는 Neue Haas Grotesk 계열
- 대제목은 크고 얇게, 본문은 가독성 중심으로 구성한다.
- 가격, 회차, 강의 번호는 숫자 타이포그래피를 적극 활용한다.

## 6. 컴포넌트 원칙

### 버튼

- Primary: black background, white text, 높이 52-60px
- Secondary: white background, black border, black text
- Disabled: gray background, gray text

### 강의 카드

- 카드 전체를 클릭 가능한 영역으로 설계한다.
- 번호, 카테고리, 강의명, 요약, 기간, 방식, 가격, 상태를 표시한다.
- 마감임박/마감은 텍스트 배지로 표현하고 색상 의존도를 낮춘다.

### 결제 CTA

- 강의 상세 데스크톱에서는 우측 sticky purchase box를 사용한다.
- 모바일에서는 하단 fixed CTA bar를 사용한다.
- CTA 문구는 `수강 신청하기`, `결제 진행하기`, `신청 완료`처럼 명확하게 쓴다.

## 7. 접근성

- 텍스트 대비는 WCAG AA 기준을 목표로 한다.
- 버튼/링크는 키보드 포커스 상태를 명확히 제공한다.
- 아이콘만으로 의미를 전달하지 않고 텍스트 라벨을 함께 제공한다.
- 결제 실패, 입력 오류는 색상뿐 아니라 문장으로 안내한다.
