# 01. 서비스 기획

## 1. 서비스 정의

INSHOW ACADEMY는 '인테리어쇼 아카데미'의 준말로, 인테리어 실무자가 실제 업무에서 바로 활용할 수 있는 강의를 신청하고 결제할 수 있는 프리미엄 교육 플랫폼이다.

## 2. 핵심 목표

- 강의 정보를 고급스럽고 신뢰감 있게 전달한다.
- 사용자가 강의 상세 페이지에서 일정 확인, 신청, 결제까지 끊김 없이 완료하게 한다.
- 운영자가 강의, 신청자, 결제, 환불을 직접 관리할 수 있게 한다.
- 향후 온라인 영상 강의와 수료증, 기업교육 문의, 멤버십으로 확장 가능한 구조를 만든다.

## 3. 주요 사용자

| 사용자 | 목적 | 주요 행동 |
|---|---|---|
| 예비 수강생 | 실무에 필요한 강의 탐색 | 강의 목록/상세 확인, 신청, 결제 |
| 현직 실무자 | 특정 업무 역량 강화 | 커리큘럼 비교, 일정 선택, 자료 다운로드 |
| 예비 창업자 | 견적/계약/운영 실무 학습 | 입문/창업 강의 신청 |
| 관리자 | 강의 운영 | 강의 등록, 정원 관리, 신청자 관리, 환불 처리 |

## 4. MVP 범위

MVP는 '강의 탐색 -> 수강 신청 -> 결제 -> 신청 완료 -> 관리자 확인'까지의 전환 흐름을 완성하는 데 집중한다.

### 포함

- 홈
- 강의 목록
- 강의 상세
- 회원가입/로그인
- 수강 신청
- 결제
- 결제 완료/실패
- 마이페이지
- 관리자 강의/신청/결제 관리

### 제외 또는 2차

- 온라인 영상 플레이어
- 진도율
- 과제 제출
- 수료증 자동 발급
- 커뮤니티
- 정기 구독 멤버십

## 5. 성공 기준

- 모바일에서 3분 이내 강의 신청과 결제가 가능해야 한다.
- 결제 성공 이후 관리자와 수강생 모두 신청 상태를 확인할 수 있어야 한다.
- 강의 상세 페이지의 핵심 정보가 첫 화면과 sticky 신청 영역에서 명확히 보여야 한다.
- 이모지형 아이콘 없이 블랙앤화이트 기반의 고급 UI를 유지해야 한다.
